
# Disease Prediction System

This is a machine learning-based web application built using Streamlit.

## Features
- Predicts possible diseases based on patient data
- Built using Random Forest Classifier
- Includes preprocessing pipeline

## Model Performance
- Accuracy: ~50%
- Educational project (not for medical diagnosis)

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas
